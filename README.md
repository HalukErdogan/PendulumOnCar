# PendulumOnCar
This is a c++ package that simulates and animates the dynamics of a pendulum on car.

# Build
To build the package with cmake,
```
mkdir build
cd build
cmake ..
make
```

# Run
To simulate the dynamics of the system,
```
cd ..
./build/simulate
```

# Animate
To animate the result of the simulation,
```
python3 animate.py
```
